# Przepisy-Qt
Projekt z realizowany jako zaliczenie na przedmiot uczelniany, Programowanie Obiektowe.

Wykorzystane technologie:
-------------------------
* C++ i Qt 5,
* SQLite jako drobna baza danych do przechowywania przepisów

Czego się nauczyłem?
-------------------------
* Był to mój pierwszy, użytkowy program napisany obiektowo,
* Pierwszy kontakt z obszerną biblioteką Qt, wybór padł na nią, bo jest znaną biblioteką linuxową głównie (KDE),
* I znowu pierwszy kontakt, baza SQLite, okazało się, że Qt posiada driver do obsługi bazy,
* Wyzwaniem było dla mnie zapisanie obrazka jako BLOB do bazy, nigdy wcześniej nie robiłem czegoś takiego,
* Sporo problemów przysporzyło mi zrozumienie jak działa komunikacja między oknami, jak między nimi przesyłać wartości,obiekty.
